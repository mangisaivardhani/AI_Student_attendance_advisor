# Agentic AI Attendance Advisor

## Objective
This project analyzes student attendance records and provides personalized recommendations using a rule-based AI agent.

## Technologies Used
- Python
- Google Colab
- Pandas
- Kaggle Dataset

## Features
- Reads attendance data
- Analyzes attendance percentage
- Generates personalized recommendations
- Saves the output as a CSV file

## Output
The project creates a new CSV file containing attendance recommendations for each student.
